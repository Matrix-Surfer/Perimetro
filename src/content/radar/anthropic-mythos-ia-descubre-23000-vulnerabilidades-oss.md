context: "Mythos, el sistema de IA de Anthropic para descubrir vulnerabilidades de seguridad, analizó 1,000 proyectos de software abierto y detectó 23,000 fallas potenciales, muchas confirmadas como críticas o de alta severidad. Anthropic extiende ahora el acceso a gobiernos y organizaciones seleccionadas mientras define los controles de uso. El impacto práctico para empresas mexicanas: la velocidad de descubrimiento de vulnerabilidades en el software que usan cotidianamente acaba de acelerarse de forma significativa. Los servidores web, gestores de contenido, bases de datos y frameworks que corren sus sistemas están siendo escaneados a una escala que antes era imposible. Mantener software actualizado dejó de ser una práctica recomendable para convertirse en la única defensa real frente a un calendario de parches que ya no puede seguir el ritmo de los hallazgos."
