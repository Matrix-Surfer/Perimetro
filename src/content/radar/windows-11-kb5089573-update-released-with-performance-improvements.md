## Resumen

Microsoft has released the KB5089573 preview cumulative update for Windows 11 versions 25H2 and 24H2, which comes with 30 changes, including performance and reliability improvements. [...]

## Relevancia

Las organizaciones con flotas de equipos en Windows 11 versiones 24H2 o 25H2 deben incorporar esta actualización en su ciclo de gestión de parches. Las mejoras de confiabilidad en estas actualizaciones acumulativas suelen incluir correcciones de seguridad que Microsoft no documenta de forma explícita hasta semanas después.
