## La señal

Con inteligencia artificial generativa, describir una aplicación en lenguaje natural ya basta para obtener una aplicación funcionando en minutos. El desarrollo pasó de meses de trabajo a una conversación.

## El supuesto que se rompe

Las prácticas de seguridad (revisión de código, modelado de amenazas, pruebas de vulnerabilidades) se diseñaron para un desarrollo de software mucho más lento, con etapas claras donde alguien podía intervenir. Cuando el código se genera en segundos, esas etapas dejan de existir.

## Qué observar

Qué aplicaciones se están creando dentro de la empresa usando herramientas de IA sin pasar por ningún tipo de revisión, y quién tiene autoridad para usarlas. Cuanto más fácil es crear software, más fácil es acumular riesgo sin que nadie lo note.
