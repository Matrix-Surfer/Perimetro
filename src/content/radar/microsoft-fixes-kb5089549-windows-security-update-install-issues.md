## Qué está pasando

La actualización de seguridad de mayo 2026 para Windows 11 (KB5089549) tuvo problemas de instalación en algunos equipos, generando el error 0x800f0922. Microsoft identificó y resolvió la causa — un problema de compatibilidad con ciertas configuraciones de sistema — y publicó la corrección. Los equipos que fallaron anteriormente pueden reintentar la instalación.

## Para organizaciones

- Si algún equipo Windows 11 mostró el error 0x800f0922 al intentar actualizar en mayo, reintentar la actualización ahora — el problema está resuelto.
- Los equipos sin la actualización de seguridad de mayo siguen expuestos a las vulnerabilidades que corrige — no aplazarla más.
- Si gestionas actualizaciones de forma centralizada, verificar que los equipos afectados recibieron la corrección en el siguiente ciclo de actualización.

## Decisión recomendada

Verificar que todos los equipos Windows 11 tienen instalada la actualización KB5089549, especialmente aquellos que mostraron errores de instalación en mayo.
