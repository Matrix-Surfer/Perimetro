## Contexto

Google publicó una actualización de emergencia para Chrome que corrige múltiples vulnerabilidades críticas. Las fallas permiten a atacantes ejecutar código arbitrario en el equipo de la víctima simplemente haciendo que visite un sitio web malicioso o comprometido. No se requiere ninguna interacción adicional más allá de abrir la página.

## Impacto potencial

Chrome es el navegador dominante en entornos empresariales. Cualquier empleado que navegue con una versión sin actualizar está expuesto — el ataque no requiere descargar nada ni hacer clic en archivos adjuntos. Un sitio legítimo comprometido puede servir como vector sin que el usuario lo detecte.

## Recomendaciones

Abrir Chrome → menú (⋮) → Ayuda → Información de Google Chrome. El navegador verificará e instalará la actualización disponible. Reiniciar Chrome para aplicarla. No requiere permisos de administrador en la mayoría de los casos.
